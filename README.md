# 🍽️ 一餐的碳足跡大冒險（Carbon Footprint of a Meal）

本專案為一個 **環境教育 × 數位學習** 的 Streamlit 教學應用，
引導學生透過「一餐的選擇」，理解 **食物、生產、包材、交通** 對碳足跡的影響。

學生可使用自己的手機或平板操作，計算結果會即時彙整成 **全班 Google Sheet 紀錄**，
供教師進行課堂討論與後續分析。

---

## 🎯 教學目標
- 理解不同食物與消費行為的碳足跡差異
- 比較「內用 / 外帶」、「是否選包材」、「交通距離」的影響
- 培養低碳消費與環境決策能力
- 練習資料視覺化與數據討論（圓餅圖、長條圖）

---

## 🧩 功能特色
- 📍 自動抓取使用者定位（或手動設定）
- 🍛 主餐隨機抽選（3 選）
- 🥤 飲料選擇（可喝 / 不喝）
- 🍰 甜點隨機給 5 選 2
- 📦 餐具 / 包材複選（可不選）
- 🛒 搜尋附近分店並以地圖點選（計算交通碳足跡）
- 📊 即時圓餅圖、長條圖呈現碳足跡比例
- 📥 個人結果可下載 CSV
- 📄 全班結果自動寫入 Google Sheet（Service Account）

---
